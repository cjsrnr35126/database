## 통계 데이터 베이스 과제

### ① 데이터 및 데이터베이스 구조 설명 (데이터, 테이블 등)
통계청 MDIS를 이용하여 사망원인 통계자료를 습득하였다. 데이터의 갯수는 약 2만6천여개이며 따라붙는 자료로는 질병사인분류코드와 나이,성별 분류코드가 있다. 이를 각각 사망원인 통계, 질병사인분류 , 나이분류 테이블로 만들었다.<br>
